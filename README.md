# Blackjack Java Miniprojekt 

## Beschreibung
Dieses Projekt ist ein einfaches Blackjack-Spiel, das in Java programmiert wurde. Der Spieler spielt gegen den Dealer und versucht, möglichst nah an 21 Punkte zu kommen, ohne diese Zahl zu überschreiten.

## Funktionen
- Grafische Benutzeroberfläche mit Java Swing
- Karten werden zufällig gezogen
- Spieler kann weitere Karten ziehen
- Dealer zieht automatisch Karten
- Gewinn, Verlust oder Unentschieden wird angezeigt
- Ass kann als 1 oder 11 gezählt werden

## Verwendete Technik
- Java
- Java Swing
- GitHub zur Versionsverwaltung

## Projektstruktur
- `Main.java` startet das Spiel
- `Gui.java` enthält die grafische Oberfläche und Spiellogik
- `Card.java` beschreibt eine einzelne Spielkarte
- `cards/` enthält die Kartenbilder

## Spielregeln
Ziel ist es, näher an 21 Punkte zu kommen als der Dealer.  
Bildkarten zählen 10 Punkte. Ein Ass zählt 11 Punkte, kann aber als 1 gezählt werden, wenn man sonst über 21 Punkte kommen würde.
